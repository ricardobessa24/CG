README (Space Shooter - Ricardo Bessa e Nuno Ferreira)

Descrição
Para avaliação na cadeira de Computação Gráfica decidimos 
elaborar a nossa versão de um dos jogos mais famosos de sempre.
O jogo consiste em eliminar todos os asteroides que nos aparecem
como uma ameaça, para isso teremos que disparar na sua direção e 
não deixar nenhum escapar.

O programa foi criado e testado com o Visual Studio Code.

Interação:
Tecla A e D - Movimentar a nave
Teclas setas - Movimentar o planeta Saturno
Espaço - Fazer a Nave disparar
